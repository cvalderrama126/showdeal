🚀 QUICK START (5 MINUTOS):

  1. cd showDeal\App
  
  2. cp .env.docker .env
  
  3. docker-compose up -d
  
  4. sleep 40  (esperar a que PostgreSQL esté listo)
  
  5. docker-compose exec app npx prisma generate
  
  6. curl http://localhost:3000/health
  
  ✅ Listo! http://localhost:3000
